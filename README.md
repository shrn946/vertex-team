# Vertex Team Carousel for Elementor

Professional team/member carousel widget for Elementor, powered by Swiper and built for WordPress.

## Overview

`Vertex Team Carousel for Elementor` adds a customizable team slider widget with:

- Member image, name, position, and description
- Dynamic social icons (Elementor Icon Library + links)
- Optional View Info button per member
- Extensive style controls (text, social icons, button, card background including gradient)
- Responsive behavior and Swiper-based carousel UI

## Requirements

- WordPress (current stable release)
- Elementor (active)
- PHP 7.4+

## Installation

1. Download or clone this plugin folder.
2. Upload it to:
   - `wp-content/plugins/team`
3. Activate the plugin in **WordPress Admin > Plugins**.
4. Open Elementor and search for widget:
   - `Team Carousel Slider`

## Usage

1. Drag **Team Carousel Slider** into your Elementor page.
2. Add/edit team members in the **Team Members** repeater.
3. Configure:
   - Social Icons (icon + link + external + nofollow)
   - View Info button toggle and link
4. Style from the **Style** tab:
   - Text
   - Card background (classic/gradient)
   - Social icon appearance
   - Button typography, colors, padding, radius, alignment
